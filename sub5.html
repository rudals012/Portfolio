프로젝트 개요

대표적인 OTT 서비스 사이트 넷플릭스를 React를 사용하여 반응형 카피앱으로 구현하였습니다.

주요 업무

- 구글의 Firebase Auth로 로그인/회원가입 및 프로필 구현
- axios를 사용하여 영화 데이터 호출
- React Router를 이용하여 리다이렉트 및 경로 이동 처리
- React Swiper를 사용해 배너 슬라이드 생성
- Styled-Components를 사용하여 일부 컴포넌트 스타일 적용
